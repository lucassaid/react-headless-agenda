# Ticks

### Using `<Ticks>` to render a markup for each hour. 

<Canvas of={TicksStories.ForHours} />

Every tick has the current `hour`, and the `top` position in pixels. `top` is calculated based on the height of the container.

 
<Source
  code={`
<Ticks>
  {({ containerRef, ticks }) => (
    <div
      className="relative h-[700px]"
      ref={containerRef}
    >
      {ticks.map(({ hour, top }) => (
        <div
          key={hour}
          className="text-slate-300 absolute"
          style={{ top: top - 14 }}
        >
          {hour} hs
        </div>
      ))}
    </div>
  )}
</Ticks>
  `}
  language="tsx"
/>


### Using `<Ticks>` to render a line for each hour

<Ticks>

      {ticks.map(({ hour, top }) => (

          className="h-0.5 bg-slate-100 absolute w-full"

        />
      ))}

</Ticks>
  `}
